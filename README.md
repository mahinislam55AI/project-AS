## 🚀 How to Use This Course

1. **Follow modules in order** — each builds on the previous
2. **Practice every exercise** before moving to the next module
3. **Simulate in TIA Portal PLCSIM** before deploying to real hardware
4. **Complete capstone projects** to solidify all concepts

---

## 🛠️ Required Tools

- **Siemens TIA Portal V16+** (STEP 7 Basic or Professional)
- **PLCSIM** (for simulation without real hardware)
- **S7-1200 CPU** (any variant — 1211C recommended for beginners)
- **24V DC Power Supply**
- **PROFINET cable (Ethernet)**
